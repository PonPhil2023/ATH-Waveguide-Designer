# ATH Waveguide Designer GUI

A modern graphical user interface for designing acoustic waveguides with **ATH (Athena)**.

This project provides a more convenient workflow for ATH by combining:

- graphical parameter editing
- automatic configuration file generation
- direct ATH execution
- embedded STL visualization
- organized simulation output management

It is intended for users who want to design and inspect waveguides more efficiently than using the ATH command-line workflow alone.

---

# ⚠ Important: Install ATH First

This application is **NOT a replacement for ATH**.

It is a **GUI frontend built on top of ATH**, meaning ATH must already be installed on your computer.

ATH is the acoustic waveguide generator developed by **Marcel Batik** and widely used in loudspeaker waveguide design.

Official website:

https://www.at-horns.eu/index.html

---

# Installing ATH

## 1. Download ATH

Download ATH from the official website:

https://www.at-horns.eu/index.html

After downloading, extract the package to a directory such as:


## 2. Test ATH Installation

Open **Command Prompt** and run:

```bash
D:\ATH\ath.exe
```

If ATH launches correctly, the installation is complete.

# Supported Waveguide Types

The current version of the program supports **three waveguide profiles** commonly used in loudspeaker design:

| Type           | Description                                          |
| -------------- | ---------------------------------------------------- |
| **OS-SE**      | Oblate Spheroid – Super Elliptical waveguide         |
| **Tritonia**   | Tritonia waveguide profile developed by Marcel Batik |
| **Tritonia-M** | Modified Tritonia profile with adjustable curvature  |

These profiles allow designers to explore different acoustic directivity and wavefront control characteristics.

---

# Waveguide Parameters (with Chinese description)

The GUI exposes several parameters used by ATH to define the waveguide geometry.

| Parameter | Meaning                     | 中文說明     |
| --------- | --------------------------- | -------- |
| **L**     | Waveguide length            | 波導長度     |
| **r0**    | Throat radius               | 喉口半徑     |
| **a0**    | Initial angle               | 初始擴散角    |
| **a**     | Flare angle function        | 擴張角函數    |
| **p**     | Normalized profile position | 正規化輪廓位置  |
| **cos()** | Cosine shaping function     | 餘弦形狀控制函數 |

These parameters control the **waveguide flare rate, curvature, and acoustic loading**.

# Example ATH Configuration

Example OS-SE configuration generated by the GUI:

```text
OSSE = {
  L = 135.0
  r0 = 36.0
  a0 = 4.2
  a = 48.5 - 7.0*cos(2.0*p)^5
```

# Installation

## 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/ATH-Waveguide-Designer.git
```

Enter the project folder:

```bash
cd ATH-Waveguide-Designer
```

---

## 2. Install Dependencies

If Python dependencies are required:

```bash
pip install -r requirements.txt
```

---

## 3. Configure ATH Path

Set the ATH executable path in the configuration file or script.

Example:

```text
ATH_PATH = "D:/ATH/ath.exe"
```

---

# Usage

## Step 1 — Launch the GUI

Start the application:

```bash
python main.py
```

---

## Step 2 — Configure Waveguide Parameters

Select the waveguide type:

* OS-SE
* Tritonia
* Tritonia-M

Then adjust the geometric parameters using the GUI controls.

---

## Step 3 — Run ATH Simulation

Press **Run Simulation**.

The program will:

1. Generate ATH configuration
2. Execute ATH
3. Produce STL geometry

---

## Step 4 — View STL

The generated waveguide geometry can be previewed in the GUI or exported for:

* CAD software
* 3D printing
* acoustic simulation

Typical export format:

```text
STL
```
